FirebaseConnectivityProvider

FirebaseConnectivityProvider class Null safety

A provider using the FirebaseConnectivityRepository.

Reports the connectivity status of the screen and notifies devices to this. The init method must be called to initialise FirebaseConnectivityProvider.
